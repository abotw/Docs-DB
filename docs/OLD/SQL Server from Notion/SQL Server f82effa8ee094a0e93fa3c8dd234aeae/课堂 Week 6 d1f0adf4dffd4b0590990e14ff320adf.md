## #3 将 E-R 图转换为关系模型（二维表）

实体 —— 表

实体间的联系 —— 表（1. 相关的实体的属性、2. 联系特有的属性）

如果联系中只有 1，联系不需要单独设置一个表，只需要在其中一个实体的属性中加上另一个相关实体的主键。

如果联系中有 2，联系需要单独设置为一个表，表中的属性包括两个相关实体的属性和联系独有的属性。（学生—课程—选修关系（选修成绩））
